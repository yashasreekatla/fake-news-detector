# Fake News Detector 🤖
Fake news detection system using Python and machine learning, leveraging text vectorization and logistic regression for classification.
A simple Machine Learning project that classifies news as Real or Fake.

## 🔧 Tech Used
- Python
- scikit-learn
- pandas

## ⚙️ How it Works
- Converts text into numerical features using CountVectorizer
- Uses Logistic Regression to classify news

```bash
python fake_news.py
